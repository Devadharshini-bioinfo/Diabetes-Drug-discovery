# 🧬 Computational Drug Discovery for Type 2 Diabetes

## 🔍 Overview

This project focuses on identifying potential therapeutic targets and phytocompounds for Type 2 Diabetes using integrated bioinformatics and molecular docking approaches.

---

## ⚙️ Workflow

GEO Dataset → DEG Analysis → PPI Network → Hub Gene Identification → ADME Screening → Molecular Docking

---

## 📊 Key Results

* 🔬 24,159 genes analyzed from GEO datasets
* 📈 7,564 differentially expressed genes (DEGs) identified
* 🧠 Top hub gene: STAT3
* 💊 1238 phytocompounds screened → 44 shortlisted (ADME)
* 🧪 Lead compound: Leucocyanidin
* 🔗 Binding affinity: **–8.1 kcal/mol**

---

## 🧪 Tools & Technologies

* R (DESeq2, ggplot2)
* STRING
* Cytoscape (CytoHubba)
* DAVID
* AutoDock Vina
* PyMOL
* SwissADME

---

## 🧬 Key Insights

* STAT3 identified as a critical regulator in insulin resistance pathway
* Leucocyanidin shows strong binding stability with active site residues
* Demonstrates effectiveness of computational pipeline in drug discovery
